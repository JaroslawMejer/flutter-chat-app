# Flutter Chat App

Chat app writen in Flutter.

## Requirements

- For Windows - https://flutter.dev/docs/get-started/install/windows
- For macOS - https://flutter.dev/docs/get-started/install/macos

## Used technologies
- Dart
- Flutter
- Firebase & Firestore
